## GitHub Pagesで公開する手順

### 方法1：Webインターフェースでアップロード

1. GitHubにログイン
2. 新しいリポジトリを作成（例：`kashihara-profile`）
3. 「Add file」→「Upload files」をクリック
4. `index.html`と`style.css`をドラッグ&ドロップ
5. 「Commit changes」をクリック
6. 「Settings」→「Pages」
7. 「Source」で「main」ブランチを選択
8. 保存後、`https://ユーザー名.github.io/リポジトリ名/`でアクセス可能
